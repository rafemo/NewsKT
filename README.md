# NewsKT
News app developed in Kotlin. You can see latest news, storage, delete and save them and also search custom news in the search field.

-------------------------------------------

###### Architecture
- MVVM

###### Network
- Retrofit

###### Database
- Room

###### Dependency injection
- Hilt

###### Images
- Glide

###### Other Android libraries
- Splash
- Couroutines
- Navigation
